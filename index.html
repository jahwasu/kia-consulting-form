📐 포트폴리오 구성 원칙
재무목표별 포트폴리오는 아래 5가지 원칙으로 설계됩니다
1
Core 비중 룩업 테이블
투자성향 × 잔여기간 조합으로 Core 비중을 직접 확정합니다. 여기에 목표 성격 보정(확정 지출 +10%p / 반확정 +5%p / 유연 0%p)을 가산하여 최종 Core 비중을 결정합니다. 상한은 95%입니다.
2
Core-Satellite 구조
Core(핵심자산)는 S&P500 + SCHD 2종으로 구성합니다. S&P500은 장기 성장 엔진, SCHD는 배당으로 안정성을 보완합니다. Satellite(위성자산)는 반도체·AI·인도 등 특정 섹터에 집중하여 초과수익을 추구합니다. 달러 직접+노출 비중은 항상 50% 이상을 유지합니다.
3
Constraint Rule (제약 원칙)
개별 Satellite ETF는 전체의 10% 이하(적극투자형↑·10년↑·유연 목표는 15%), Satellite 합계는 50% 이하(공격투자형·10년↑·유연은 65%)로 제한합니다. S&P500+QQQM 합계는 위험중립형 이하 50%, 적극·공격형 60% 이하입니다. 비트코인·금·저변동성 ETF는 편입 금지입니다.
4
Location Rule (계좌 배치 원칙)
절세계좌(ISA·연금저축·IRP)에는 국내상장 해외ETF(TIGER·SOL·KODEX 해외계열)만 배치하고, 일반계좌에는 미국 직구 ETF를 배치합니다. TIGER 미국반도체NV는 국내상장 ETF이므로 ISA·연금저축·IRP 모두 편입 가능합니다. IRP는 납입액의 30% 이상을 TDF(안전자산)로 의무 배정합니다.
5
Glide Path (목적자금 단계 전환)
확정 지출 목표(결혼·이사·전세 등)는 잔여기간에 따라 채권 듀레이션을 단계적으로 단축합니다. 7년↑ 미국30년국채 → 5~7년 국고채3년 → 3~5년 단기채권 → 1~3년 단기통안채 → 1년↓ KOFR MMF(100%)로 전환하여 원금을 보호합니다.
